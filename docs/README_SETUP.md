## Próximos Pasos

Para crear la estructura completa de Angular, ejecuta:

```bash
# Si tienes Angular CLI instalado globalmente
ng new control-gastos --skip-git --routing --style=css

# O si prefieres crear la estructura manualmente, los archivos necesarios son:
# - angular.json
# - src/main.ts
# - src/index.html
# - src/app/app.component.ts
# - src/app/app.module.ts (o usar standalone)
# - src/styles.css
```
