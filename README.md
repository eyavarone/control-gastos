# Control Gastos

Aplicación web para control de gastos domésticos.

## Stack Tecnológico

- **Frontend**: Angular
- **Backend**: .NET 9 (C#)
- **Base de Datos**: SQLite (desarrollo) / PostgreSQL (producción)

## Arquitectura

- **Backend**: Domain-Driven Design (DDD) con separación en proyectos (Web, Domain, Infrastructure)
- **Frontend**: Estructura modular de Angular

## Reglas de Desarrollo

Este proyecto sigue un conjunto de reglas definidas en `.cursorrules` que incluyen:
- Clean Code
- Convenciones de nomenclatura
- Testing (80% cobertura mínima)
- Documentación en código
- Manejo de excepciones centralizado
